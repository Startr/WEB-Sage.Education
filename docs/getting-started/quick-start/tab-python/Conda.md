
# Install with Conda

1. **Create a Conda Environment:**

   ```bash
   conda create -n sage-open-webui python=3.11
   ```

2. **Activate the Environment:**

   ```bash
   conda activate sage-open-webui
   ```

3. **Install Sage WebUI:**

   ```bash
   pip install sage-open-webui
   ```

4. **Start the Server:**

   ```bash
   sage-open-webui serve
   ```
